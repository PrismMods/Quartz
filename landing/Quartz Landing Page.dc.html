FEATURES

Everything, in one mod.

Nine upgrades that make Quartz worth installing.

In-Game Settings Menu

A custom uGUI menu replaces UnityModManager's default IMGUI screen — cleaner, faster, and built to actually fit the game.

Live Key Viewer

Real-time input display with a timing graph, built for streaming and practice runs.

Combo & Judgement HUD

Always-on overlays for your combo count and hit judgements.

Discord Rich Presence

Shows the level and progress you're on, live, right on your profile.

Auto Deafen

Automatically deafens your Discord voice call the moment a level starts.

Custom Themes

Recolor planets and the Otto icon to make the game feel like yours.

Nostalgia Mode

Flip back to the game's classic, pre-update look, anytime.

Editor Upgrades

BGA support and a live floor-count readout for level creators.

Plays Nice With Others

Interop with XPerfect, plus automatic import of your existing UMM settings.

2

Drop Quartz.dll into your Mods folder.

3

Install UnityModManager and point it at your game folder.

2

Grab Quartz from the UMM mod list or GitHub releases.

3

Enable Quartz in the UMM launcher, then start the game.
